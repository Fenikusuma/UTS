"Kegiatan Praktikum SIG WEB"

Praktikum Minggu 1

Kegiatan Praktikum pada minggu pertama dilakukan dengan membuat pemrograman sederhana menggunakan CSS, HTML, dan Javascript. Pada minggu pertama, dikenalkan dengan cara membuat heading, paragraf, menambahkan link, image, dan hal-ha; lainnya dengan menggunakan HTML.

Pada akhir pembelajaran terdapat tugas dengan membuat website sesuai dengan tampilan halaman HTML. Selanjutnya, mahasiswa diminta untuk membuat script untuk menampilan hasil yang sesuai seperti pada contoh gambar.

Praktikum Minggu 4

Kegiatan praktikum minggu keempat dilakukan dengan belajar menggunakan Javascript. Hal-hal yang dipelajari meliputi penggunaan form, membaca data input, menggunakan fungsi dan lain-lain. Pada akhir pembelajaran terdapat latihan mandiri yang dapat mengasah pengetahuan tentang Javascript

Praktikum Minggu 5

Kegiatan pada minggu kelima ini adalah pengenalan bootstrap. Berisi tentang komponen halaman dari web, konsep dan komponen dari bootstrap itu sendiri. Pada akhir kegiatan, mahasiswa diminta untuk membuat CV dari template bootstrap

Praktikum Minggu 6

Kegiatan minggu keenam ini dilakukan dengan belajar tentang Leaflet dan memasukkan data GeoJSON. Mahasiswa diminta untuk membuat leaflet masing-masing dengan menampilkan point dan penggunaan plugin pendukung.
